# Third Party Notices

Bucket Studio is built with Flutter and Dart and depends on open-source packages from the Flutter/Dart ecosystem.

Runtime dependency notices should be reviewed before every public release. The private source repository contains the authoritative dependency lockfile for each build.

## Private UI Package

Bucket Studio uses a private company UI package in compiled form. The package source is not redistributed with public beta releases.

## Fonts And Assets

Bucket Studio includes bundled fonts and branding assets. Confirm each asset's redistribution rights before publishing release artifacts.

## Release Practice

Each public release should include:

- The application artifact for each platform.
- `SHA256SUMS.txt`.
- This notice file.
- `PRIVACY.md`.
- `SECURITY.md`.
- `LICENSE.txt`.
